Daily Code Practice
A simple daily habit: solve one small coding problem, save it in a new dayXX/ folder, commit, and push. Over time this builds a strong GitHub contribution history and a real record of practice.

How to use this repo
Each day, create a new folder: dayXX (e.g. day02, day03, ...)
Add a problem.md describing the problem (optional but nice)
Add a solution.py with your code and a couple of test prints

Day 1: Prime Check

Examples: is_prime(7) -> True is_prime(10) -> False is_prime(1) -> False

How to run
