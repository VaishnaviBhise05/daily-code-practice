# Daily Code Practice

A simple daily habit: solve one small coding problem, save it in a new
`dayXX/` folder, commit, and push. Over time this builds a strong GitHub
contribution history and a real record of practice.

## How to use this repo

1. Each day, create a new folder: `dayXX` (e.g. `day02`, `day03`, ...)
2. Add a `problem.md` describing the problem (optional but nice)
3. Add a `solution.py` with your code and a couple of test prints
4. Commit and push:

```bash
git add .
git commit -m "Day 2: <short description>"
git push
```
