## 📅 Day 1: Prime Number Check

### Problem

Write a function `is_prime(n)` that returns:

- `True` if the number is prime.
- `False` otherwise.

### Example

```python
is_prime(7)   # True
is_prime(10)  # False
is_prime(1)   # False
```

